# simpletools

Biblioteca simples de utilitários matemáticos — Projeto M3.

## Instalação

```bash
pip install -i https://test.pypi.org/simple/ simpletools
```

## Quickstart

```python
from simpletools import add, div, mean, factorial

print(add(2, 3))
print(div(10, 2))
print(mean([1, 2, 3]))
print(factorial(5))
```
